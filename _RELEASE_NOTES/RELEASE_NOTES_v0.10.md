# リリースノート - Version 0.10

2026/01/08

## Version 0.10

### UIレイアウトの刷新

- **ツールボタンの縦並び配置**
  - ヘッダーのツールボタン（余白最適化、フォント設定、言語メニュー、PWA設定、ショートカット一覧）を縦並びに変更
  - ハンバーガーボタン（ツールバー表示/非表示）の上に配置
  - 画面右下に統一されたボタン配置を実現

- **ツールボタンのデザイン統一**
  - すべてのツールボタンを丸形（40px × 40px）に統一
  - ハンバーガーボタンと同じサイズとスタイル
  - モバイルでは44px × 44pxに拡大

- **ハンバーガーボタンの独立化**
  - ツールバー表示/非表示ボタンをツールボタンとは別処理に変更
  - 元の位置（右下）に独立して配置
  - アイコンの表示のみが変わる仕様を維持

### コード改善

- **クラス名のリネーム**
  - `fullscreen-toggle` → `toolbar-toggle` に変更
  - より適切な命名に統一
  - 関連するIDも `fullscreenToggle` → `toolbarToggle` に変更

### ナビゲーション改善

- **ナビゲーションブロックの位置調整**
  - フッター（サイドバー開閉ボタン、検索フィールド、前の詩歌、次の詩歌）を下部より少し上に移動
  - `bottom: 8px` → `bottom: 20px` に変更
  - より操作しやすい位置に配置
