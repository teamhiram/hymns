# リリースノート - Version 0.2

2026/01/05

## Version 0.2

### 新機能
- **全訳（Zennyaku）表示機能**
  - ヘッダーに「全訳」トグルボタンを追加
  - ONの場合、デスクトップでは歌訳の右側に表示
  - モバイルでは各節ごとに交互に表示
  - 全訳テキストは視覚的に区別されるスタイリング（ボーダー、パディング）

- **中国語（Chinese）表示機能**
  - ヘッダーに「中文」トグルボタンを追加
  - 全訳と同様の表示ロジック（デスクトップ：横並び、モバイル：交互表示）
  - `chinese_hymns_data.json`から中国語データを読み込み

- **フォント設定メニューの追加**
  - ヘッダーに「Aa」ボタンを追加
  - フォント選択：ゴシック（GlowSans Normal）、細長（GlowSans Compressed）、明朝（Hiragino Mincho ProN）
  - 節内行間の調整（1.2〜2.5）
  - 節行間の調整（4px〜24px）
  - フォントサイズの調整（0.8em〜1.5em）
  - ウェイト選択：細字（300）、中字（400）、太字（500）
  - 透明度設定：80%、90%、不透明（設定メニュー自体の透明度）
  - 設定はlocalStorageに保存され、次回訪問時も維持されます

- **データ統合**
  - `utayaku`と`zennyaku`データを`hymns_data.json`に統合
  - 中国語データを`chinese_hymns_data.json`として追加

### UI改善
- 検索結果のプレビューを2emインデント
- サイトタイトルを「聖歌集」から「詩歌集」に変更
- サイドバーヘッダーを「詩歌」から「検索結果」に変更
- コンテナをフル幅に変更（`width: 100%`, `margin: 0`, `border-radius: 0`）
- 全訳・中国語テキストの色をグレーから黒に変更
- 全訳・中国語のフォントサイズを歌訳と同じサイズに統一
- フォントプレセットボタンのスタイル調整
  - paddingを削減（6px 12px → 4px 8px）
  - ラベルとボタンのフォントサイズを縮小（0.85em → 0.75em）
  - 改行防止のため`white-space: nowrap`を追加
- 節内行間の設定を全訳・中国語にも適用
- 全訳・中国語テキストの色を完全な黒（#000000）に変更
- 全訳により多くの幅を与えて折り返し改行を防ぐ（flex: 1.5）
- 「中文」と「全訳」トグルボタンをデフォルトでONに設定

### データ処理
- `japanese-hymnal-zennyaku.txt`から作者名を抽出・削除
- 詩歌J1001-J1007をJ780の後に移動
- `japanese-hymnal-utayaku.txt`と`japanese-hymnal-zennyaku.txt`を再パース
- `chinese-hymnal.txt`をパースしてJSON形式に変換

### バグ修正
- 透明度設定が歌詞ではなく設定メニューに適用されるように修正
