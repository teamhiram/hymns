# リリースノート - Version 0.4

2026/01/07

## Version 0.4

### 新機能
- **フッターの追加**
  - 画面下部に固定フッターを追加
  - 検索フィールドをフッターに配置（常にアクセス可能）
  - サイドバーの開閉ボタンをフッターに配置
  - 前後の詩歌へのナビゲーションボタンをフッターに配置
  - モバイル表示ではサイドバーがフッターの上に表示されるように調整

### UI改善
- **歌詞のフォントサイズを拡大**
  - ベースフォントサイズを0.95emから1.1emに変更
  - フォントサイズが100%の時でも、より読みやすい大きさで表示されます

- **フォントサイズスライダーの範囲拡大**
  - 最大値を150%（1.5em）から200%（2.0em）に引き上げ
  - より大きなフォントサイズまで調整可能になりました
