# リリースノート - Version 0.9

2026/01/08

## Version 0.9

### フォント設定の新機能

- **左右余白の調整**
  - フォント設定メニューに「左右余白」スライダーを追加
  - レイアウト設定の下に配置
  - 範囲: 0px ～ 96px（ステップ: 2px）
  - デフォルト: 0px（既定の12pxに加算）
  - モバイルでは10px基準に加算

- **リセットボタン**
  - フォント設定メニューの最後にリセットボタンを追加
  - すべてのフォント設定をデフォルト値に戻す機能
  - 設定はlocalStorageにも反映

- **デフォルトフォントサイズの変更**
  - デフォルトフォントサイズを100%から110%に変更
  - より読みやすい初期設定に

### モバイルUI改善

- **ヘッダーボタンのサイズ拡大**
  - スマホでのヘッダーボタンサイズを24px × 24pxから44px × 44pxに拡大
  - タップしやすさを向上

- **検索窓の幅拡大**
  - スマホでの検索窓の幅を1.5倍に拡大
  - より入力しやすいサイズに

- **サイドバーの視覚改善**
  - スマホでのサイドバー上部の細い線を削除
  - よりクリーンな見た目に

### UIデザインの統一

- **Rボタンのホバースタイル**
  - ホバー時の背景色を紫から白に変更
  - 文字色を青（#667eea）に統一
  - 他のボタン（前の詩歌/次の詩歌）とスタイルを統一

- **ヘッダーボタンの線の太さ統一**
  - SVGアイコンのstroke-widthを2から1.5に変更
  - ボタンのボーダー（1px）と視覚的に統一
