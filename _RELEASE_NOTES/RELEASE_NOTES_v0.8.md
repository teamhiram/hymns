# リリースノート - Version 0.8

2026/01/08

## Version 0.8

### フォント設定の拡張

- **行間の範囲拡張**
  - 最小値を1.2から1.0に変更
  - より密な行間設定が可能に

- **フォントサイズの範囲拡張**
  - 最大値を200%から250%に変更
  - より大きなフォントサイズに対応

### キーボードショートカットの変更

- **言語メニューのショートカット追加**
  - `L`キーで言語メニュー（地球アイコン）の開閉が可能に
  - レイアウト切替のショートカットを解除（メニューから操作）

- **ショートカット一覧のラベル改善**
  - `S`: 「検索」→「検索欄に移動」
  - `W`: 「ウェイト切替」→「フォントウェイト切替」
  - `L`: 「言語メニュー」→「言語メニューの開閉」
  - `I`: 「ショートカット一覧(本メニュー)」→「本メニューの開閉」

### UI改善

- **言語メニューの名称統一**
  - 「訳本メニュー」を「言語メニュー」に統一
  - 地球アイコンボタンのツールチップに「言語メニュー (L)」を表示

### キーボードショートカット一覧（更新）

| キー | 機能 |
|------|------|
| ← | 前の詩歌 |
| → | 次の詩歌 |
| J | 歌訳の表示/非表示 |
| Z | 全訳の表示/非表示 |
| C | 中文の表示/非表示 |
| L | 言語メニューの開閉 |
| O | 余白最適化 |
| S | 検索欄に移動 |
| R | 検索結果(Result)の開閉 |
| ↑ | フォント拡大 |
| ↓ | フォント縮小 |
| F | フォント切替 |
| W | フォントウェイト切替 |
| T | ツールバー表示/非表示 |
| I | 本メニューの開閉 |
